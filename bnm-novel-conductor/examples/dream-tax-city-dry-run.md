# Example Dry Run: Dream Tax City

## User Request

我想写一个城市里梦境会被征税的故事，主角是一个替别人偷回梦的人，整体想要悬疑一点，但不是纯探案。

## Conductor Classification

- Layer: concept, genre, premise, worldbuilding
- Mode: professional
- Genre pack: mystery-thriller, literary speculative, possible urban fantasy
- Missing artifacts: all core artifacts
- Safe to proceed: yes, with working assumptions

## Internal Routing

- Genre Strategist: reader promise and genre boundary
- Premise Doctor: story engine and long-form viability
- Worldbuilding Editor: rules, costs, institutions
- Character Architect: protagonist pressure
- Story Architect: first dramatic question

## User-Facing Response Pattern

这不是单纯的悬疑点子，而是一个带悬疑驱动的都市幻想设定。风险是：梦境征税很有新鲜感，但如果只停留在设定，故事会变成概念展示。必须马上定义主角为什么非偷梦不可、偷梦会付出什么代价、谁从梦税制度中获利。

我会先生成轻量 `story-brief` 和 `world-rules-light`。这些不是最终 canon，里面标注为“待作者确认”的设定后续需要你拍板。
