# BNM Setup

Use this setup skill to register BMad Novel Method module defaults.

BNM is a multi-skill module for long-form fiction development. Its primary user-facing entry point is `bnm-novel-conductor`; specialist workflows may be invoked directly by advanced users.

## Setup Responsibilities

- Register module identity.
- Register visible commands.
- Establish default output folder for novel artifacts.
- Establish default manuscript language.
- Establish default operating mode.

## Expected Artifacts

- `assets/module.yaml`
- `assets/module-help.csv`

## Default Recommendation

Use `professional` mode unless the user explicitly asks for fast brainstorming or strict studio enforcement.
