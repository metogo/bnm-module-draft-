# BNM Architect Plot

## Metadata

- code: bnm-architect-plot
- title: Architect Plot
- description: Builds plot architecture, turning points, causality, subplot function, and reveal obligations.

## Purpose

Turn story bible, characters, and world constraints into a consequence chain.

## Roles Used

- Story Architect
- Character Architect
- Genre Strategist
- Pacing Doctor
- Continuity Editor

## Flow

1. Select structure according to the book's needs.
2. Define beginning pressure, midpoint reframe, crisis, climax, and ending.
3. Map character arc turns to plot turns.
4. Remove ornamental subplots.
5. Record reveal order and foreshadowing obligations.

## Output

- Plot architecture
- Act breakdown
- Turning points
- Subplot map

## Gate

A valid outline is a chain of cause, choice, and consequence. Chronology is not enough.
