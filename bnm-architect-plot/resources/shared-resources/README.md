# Shared Resources

These resources are bundled with the module draft so it can be moved or installed without depending on the outer specification folder.

## Contents

- `role-contract.md`: common behavior contract for BNM specialists.
- `anti-ai-prose-rules.md`: prose rules for reducing generic AI writing.
- `anti-ai-prose-checklist.md`: concrete red flags and repair moves.
- `evaluation-rubric.md`: professional quality scoring standard.
- `chinese-output-policy.md`: Chinese user-facing tone and canon-status policy.
- `templates/`: durable project artifact templates.
- `genre-packs/`: genre-specific craft constraints and routing notes.

## Usage

Specialist skills should use these resources as reference material when available. The Novel Conductor should prefer project-specific artifacts over generic templates, but use templates to create lightweight missing documents when the user has not prepared them.
