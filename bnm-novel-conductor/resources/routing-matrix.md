# Routing Matrix

## Idea / Premise

Use:

- Genre Strategist
- Premise Doctor
- Theme Critic when thematic pressure is visible

Output:

- Story brief
- Premise diagnosis
- Reader promise

## Character Problem

Use:

- Character Architect
- Dialogue Editor when voice is involved
- Developmental Editor when motivation affects plot

Output:

- Character diagnosis
- Relationship pressure notes
- Arc repair plan

## Worldbuilding

Use:

- Worldbuilding Editor
- Continuity Editor
- Story Architect if rules affect plot

Output:

- Rules and costs
- Setting conflict map
- Canon notes

## Chapter Drafting

Use:

- Chapter Planner
- Continuity Editor
- Scene Writer
- Dialogue Editor as needed
- Style Guardian

Output:

- Chapter brief
- Scene cards
- Draft
- Ledger updates

## Revision

Use:

- Developmental Editor
- Pacing Doctor
- Continuity Editor
- Style Guardian only after structural diagnosis

Output:

- Failure layer
- Reader-impact diagnosis
- Rewrite plan
